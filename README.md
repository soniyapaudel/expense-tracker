# Expense Tracker (CLI)
A simple CLI-based Expense Tracker application built using Java and Maven 
This project helps user track daily expenses, view totals, and filter expenses by category.


# Features 
-> Add new Expense 
-> View all Expense 
-> Calculate Total Expense 
-> Filter expenses by Category 
-> Clean Maven project Structure 

# Tech Stack 
-> Java
-> MAven 
-> CLI(Scanner-based input )
